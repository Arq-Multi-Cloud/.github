# Arq Software — Portable Multi‑Cloud Backup Utility

## About Arq Software

Arq Software provides professional encrypted backup to virtually any cloud storage service or remote server. Arq Software supports AWS S3 (and S3‑compatible like MinIO, Wasabi), Google Drive, Microsoft OneDrive, Dropbox, Backblaze B2, SFTP, WebDAV, and network volumes. Arq Software combines client‑side encryption (AES‑256 with user‑controlled password), versioned backups (keep all changes), deduplication (block‑level), compression, and scheduled backups. Arq Software enables restoring to any point in time, browsing backup contents, mounting backups as disks, and restoring individual files or entire disks. Arq Software integrates backup plan dashboard, activity log, and notification directly into the interface. Arq Software enables individuals and businesses to implement 3‑2‑1 backup strategy without vendor lock‑in. Arq Software runs on portable storage, USB drives, and local user folders. Arq Software is trusted by security‑conscious users, system administrators, DevOps engineers, business owners, and photographers worldwide.
